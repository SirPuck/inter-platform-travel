# What is this ?

This is the standalone travel function from IPL: Inter Platform Logistics.
Since most of IPL functionalities are implemented directly inside the game in 2.1, the travel part gets its own mod.

# How does this work ? 

When you are inside a platform hub, press CTRL + T, a gui will open.
You can travel towards other space platforms at the same space location.
You can change the keybind in the controls options.
